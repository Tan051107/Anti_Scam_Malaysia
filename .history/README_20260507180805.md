# Anti_Scam_Malaysia
=======
# 🛡️ Anti-Scam Malaysia

AI-powered scam detection and education platform for Malaysians.
Platform pengesanan dan pendidikan penipuan berkuasa AI untuk rakyat Malaysia.

---

## Features / Ciri-ciri

| Feature | Description |
|---------|-------------|
| 🔍 **Analysis Bot** | Analyze suspicious messages, URLs, phone numbers, and images for scam indicators |
| 🎮 **Scam Simulator** | Practice identifying real Malaysian scam scenarios in a safe environment |
| 📋 **Report Generator** | Generate structured incident reports for PDRM / BNM submission |

---

## Project Structure

```
anti-scam-malaysia/
```

## Quick Start

### Prerequisites
- Node.js 18+ and npm
- Python 3.10+

---

### Backend Setup

```bash
cd anti-scam-malaysia/backend

# Create virtual environment
python -m venv venv

# Activate (Windows)
venv\Scripts\activate

# Activate (macOS/Linux)
source venv/bin/activate

# Install dependencies
pip install -r requirements.txt

# Copy environment file
cp .env.example .env

# Run the server
python main.py
# OR
uvicorn main:app --reload --port 8000
```

Backend will be available at: **http://localhost:8000**
API docs (Swagger): **http://localhost:8000/docs**

---

### Frontend Setup

```bash
cd anti-scam-malaysia/frontend

# Install dependencies
npm install

# Start development server
npm run dev
```

Frontend will be available at: **http://localhost:5173**

## API Endpoints

| Method | Endpoint | Description |
|--------|----------|-------------|
| `GET` | `/api/health` | Health check |
| `POST` | `/api/analysis/chat` | Analyze text message |
| `POST` | `/api/analysis/upload` | Analyze uploaded image |
| `POST` | `/api/simulator/chat` | Scam simulator chat turn |
| `POST` | `/api/simulator/reset` | Reset simulator session |

### Example: Analysis Chat

```bash
curl -X POST http://localhost:8000/api/analysis/chat \
  -H "Content-Type: application/json" \
  -d '{"message": "Your Maybank account has been suspended. Send OTP to verify: 03-1234567"}'
```

Response:
```json
{
  "reply": "🚨 CRITICAL WARNING...",
  "risk_score": 95,
  "risk_level": "CRITICAL",
  "indicators": ["Requesting OTP...", "Bank impersonation..."],
  "confidence": 92,
  "session_id": "uuid-here"
}
```
